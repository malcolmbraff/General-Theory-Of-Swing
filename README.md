# Generalized Theory of Swing

**Malcolm Braff** — Jazzcampus, HSM Basel, FHNW  
*Working paper — Rhythm Class III Compendium*

## About

This repository contains the working document for a generalized, 
formally grounded theory of swing. Rather than treating swing as 
a stylistic attribute of jazz, this theory defines it as a structural 
property of rhythmic subdivision: swing arises whenever the subdivision 
grid of a beat is distorted such that its constituent intervals become 
unequal.

From this foundation, the theory develops a comprehensive framework 
covering groove balance, rhythmic consonance and synchronicity, 
potential energy, Euclidean rhythms, and the concept of the **Quplet** 
— a privileged rotation of a Euclidean rhythm that maximizes entropy 
and minimizes potential energy.
